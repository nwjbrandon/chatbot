# Backend
Common Web Services

## Install
- External dependencies
```
sudo apt install python3.6 python3-pip redis
```
- Python libraries
```
python3.6 -m pip install virtualenv
python3.6 -m virtualenv env
source env/bin/activate
pip install -r requirements.txt
```

## Run
- Run on local machine 
```
./start_main_app.sh
```
